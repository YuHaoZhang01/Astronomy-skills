# 数据源与筛选规则

## 检索顺序

1. NASA ADS
2. 论文正式页面或 arXiv 页面
3. 出版社补充材料页面
4. CDS / VizieR 机器可读表
5. 论文明确链接的 Zenodo / Figshare / Dataverse / 项目仓库
6. 论文明确指向的官方科学档案，例如 MAST、HEASARC、IRSA、ESO

## 可接受的数据来源

- 论文正文、补充材料或数据可得性声明里明确给出的下载地址
- 论文引用的可信科学档案记录
- 与论文一一对应的机器可读表
- 原作者公开发布、并能和论文直接对应的数据仓库记录

## 不可接受的数据来源

- 只包含图片或网页表格，但没有原始文件下载
- 论坛、博客、未知镜像或第三方“整理版”
- 来源与论文没有直接对应关系的数据包
- 需要账户审批、邮件索取或人工申请的数据

## 主题检索提示

- 先用天体类型或事件类型做主检索词，再加数据类型词。
- 常见数据类型词：
- `light curve`
- `photometry`
- `spectra`
- `machine-readable table`
- `supplementary data`
- `catalog`
- 对缩写、别名和拼写变体都要检索。例如：
- `Type Ibn`
- `SN Ibn`
- `Ibn supernova`

## 选择规则

- 需要论文清单时，先做宽搜，再按是否存在公开数据收缩。
- 需要下载时，不要假设“这类论文通常会公开数据”；必须打开实际页面确认。
- 同一论文同时提供补充材料和档案镜像时，优先保留论文原始补充材料，再把档案链接记录到参考文献。
- 如果一篇论文只有汇总图、没有公开表，不把它算作“已获得数据”。
